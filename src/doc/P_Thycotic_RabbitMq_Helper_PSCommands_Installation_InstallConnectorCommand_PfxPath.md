# InstallConnectorCommand.PfxPath Property 
 _**\[This is preliminary documentation and is subject to change.\]**_

Gets or sets the PFX path. This could be a self-signed or a certificate from a public CA. If self-signed, the certificate should be installed on all client/engine machines. It does NOT to be installed on the RabbitMq node.
